Plans

予約できる3つのカテゴリ

最初の画面では、カテゴリ比較と導線づくりを優先しています。

1. 人力車体験

水間の空気を、ちょうどよく味わう。

まちの空気や会話を楽しみながら巡る、誠屋の定番体験です。

  • ゆっくり水間コース / 3,000円 / 約15分
  • しずかな川沿いコース / 4,000円 / 約20分
  • たっぷり水間コース / 6,000円 / 約35分
人力車の予約導線へ

2. 記念日フォトプラン

節目の一日を、写真として丁寧に残す。

人力車体験と相性のよい、自然体の撮影プランです。

  • はじまりの一枚 / 19,800円
  • 笑ったひととき / 29,800円
  • 物語を残す / 59,800円
フォトの予約導線へ

3. 記念日ムービープラン

その日の余韻まで、映像で未来へ残す。

体験の空気感や言葉にならない表情まで、短い映像にまとめます。

  • ライトプラン / 19,800円
  • スタンダードプラン / 29,800円
  • プレミアムプラン / 59,800円
